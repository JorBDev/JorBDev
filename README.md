## 🚀 Featured Projects

### 🏋️ [GymControl Desktop V1](https://porfolio-jorbdev.vercel.app/) — *Commercial Java Desktop App*
> `Java 21` · `JavaFX` · `H2` · `OpenCV` · `License3j` · `Maven`

A gym management system built as a **real product** — not a portfolio demo. Features a custom hardware-bound DRM system (RSA + SHA-256 + hardware fingerprint), real-time reactive UI with JavaFX Property Bindings, and a zero-config installer with embedded JRE via `jpackage`.

- 🔐 **DRM by hardware:** RSA key pair + SHA-256 fingerprint of motherboard serial + CPU ID — invalid on any other machine
- ⚡ **10,700+ LOC** across 59 Java classes and 17 FXML views
- 📦 **Zero dependencies for the end user:** `.exe` installer bundles JRE — no Java install required
